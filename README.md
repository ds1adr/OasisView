# OasisView
OASIS file viewer

1. OASIS: Open Artwork System Interchange Standard. (SEMI P-39, 3626)
2. Qt Project
3. C++ 17 or more.
4. FFTW Libaray
  Mac OS
    > brew install fftw
  Linux
    > sudo apt update
    > sudo apt install libfftw3-dev
5. WIP now. Currently parse Rectangle, Trapezoid, CTrapezoid, Text, Placement, Property in the Cell. (6 element types so far)
