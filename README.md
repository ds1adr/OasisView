# OasisView
OASIS file viewer

1. OASIS: Open Artwork System Interchange Standard. (SEMI P-39, 3626)
2. Qt Project
3. C++ 17 or more.
4. WIP now. Currently parse Rectangle, Trapezoid, CTrapezoid, Text, Placement, Property in the Cell. (6 element types so far)
